# Fira Code

**Fira Code** 是一种免费的等宽字体，具有编程连字功能。

更多信息请查看上游网站：https://github.com/tonsky/FiraCode

版本：6.2

## 选择哪种字体？

### 概述

- 选择你的字体家族：
  - 如果你受限于等宽字体（因为你的终端等原因），那么选择带有 `Nerd Font Mono`（或 `NFM`）的字体。
  - 如果你想要更大的图标（通常是普通字母宽度的1.5倍），选择不带 `Mono` 的字体，即 `Nerd Font`（或 `NF`）。大多数终端支持这种字体，但可能会有所不同。
  - 如果你在比例环境中工作（GUI元素或编辑演示文稿等），选择带有 `Nerd Font Propo`（或 `NFP`）的字体。

### 连字

修补后的字体通常会保留连字功能。
Nerd Fonts `v2.0.0` 在 `Nerd Font Mono` 字体中没有连字，但在 `v2.1.0` 中已取消这一限制。
如果你的终端支持连字但你不想使用连字，你可以（通常）在终端设置中禁用它们。

### 说明

一旦你确定了你的字体家族（`Droid Sans`、`Inconsolata` 等）和样式（`粗体`、`斜体` 等）选择，你有两个主要选择：

#### `选项1：下载已修补的字体`

- 从[发布页面](https://github.com/ryanoasis/nerd-fonts/releases)下载稳定版本的字体包
- 或从这里的文件夹下载开发版本

#### `选项2：修补你自己的字体`

- 使用字体修补程序提供的各种选项修补你自己的变体（例如，不包含所有符号以获得更小的字体大小）
